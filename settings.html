Coyote 3.0 Control v1.0
When enabled, the AI will be able to control your DG-LAB Coyote 3.0 e-stim device during chats.

Setup Instructions

  1. Make sure your Coyote 3.0 device is powered on and paired to the DG-LAB app on your phone
  2. Run the DG-LAB socket server locally (or use a hosted one)
  3. Click "Connect to Socket Server" below
  4. Scan the QR code with the DG-LAB app (Game Mode / Socket)
  5. Wait for Status: Paired
  6. Toggle "Enable Coyote 3.0 Control" above

Device Connection

Status: Not Connected
Default: ws://localhost:9999. Change if your socket server runs on a different host or port.
