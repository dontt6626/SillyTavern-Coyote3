Coyote 3.0 Control v1.0
When enabled, the AI will be able to control your DG-LAB Coyote 3.0 e-stim device during chats.

Connection Mode

Web Bluetooth: Direct browser-to-device. Simplest setup. Requires Chrome/Edge and the device nearby.
Socket Server: Uses a relay server + DG-LAB app. More complex but works remotely and doesn't require the browser to support Bluetooth.

Setup Instructions

  1. Make sure your Coyote 3.0 device is powered on and nearby
  2. Use Chrome or Edge (Web Bluetooth is not supported in Firefox/Safari)
  3. Click "Pair with Device" below
  4. Select 47L121000 from the Bluetooth picker
  5. Wait for Status: Paired
  6. Toggle "Enable Coyote 3.0 Control" above
  1. Make sure your Coyote 3.0 device is powered on and paired to the DG-LAB app on your phone
  2. Run a DG-LAB socket server locally (or use a hosted one)
  3. Install the server plugin (coyote3.mjs into SillyTavern plugins/ folder)
  4. Restart SillyTavern with enableServerPlugins: true
  5. Click "Connect to Socket Server" below
  6. Scan the QR code with the DG-LAB app under Game Mode / Socket Control
  7. Wait for Status: Paired
  8. Toggle "Enable Coyote 3.0 Control" above
